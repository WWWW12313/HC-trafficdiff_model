# 2026-05-19 实验日志

## CausalDiffTab v2 当前结果同步

### 1. v2 域内 benchmark 结论

- 数据集：`nyc_crash_2024_v2`，45 列 schema，full tier，10000 synthetic samples。
- 当前最强域内 TSTR 仍是 `ablation_no_causal_v2`：TSTR=0.7394，Best=0.7482，R2=0.615，MSE=0.283，WD=0.2101，JS=0.0094。
- `baseline_tabddpm_v2`：TSTR=0.7294，WD=0.4076，JS=0.0100。
- 原完全体 `macro_soft_2024_v2`，lambda=0.3：TSTR=0.7083，WD=0.4158，JS=0.0095。
- 诊断：原完全体被无因果消融反超，说明旧 soft mask + lambda=0.3 对 TSTR 有明显压制。

### 2. lambda 灵敏度实验

- `macro_soft_lam005_v2`：TSTR=0.7033，Best=0.7435，R2=0.609，MSE=0.288，MAE=0.197，WD=0.2101，JS=0.0078。
- `macro_soft_lam01_v2`：TSTR=0.6856，Best=0.7481，R2=0.615，MSE=0.283，MAE=0.205，WD=0.2101，JS=0.0102。
- 结论：单纯降低 lambda 没有解决 TSTR 下滑；lambda=0.05 改善分布保真度和 MAE，但 TSTR 仍低于无因果消融。

### 3. Opt-C：稀疏宏观因果图 + 延迟退火

- 新实验：`macro_sparse_anneal_v2`。
- 因果图只保留宏观机制边：`time -> weather`、弱 `geo -> weather`、强 `geo -> road context`、`weather/road/vehicle/time-light proxy -> crash type`、`crash type/road/vehicle/weather -> injury outcome`。
- 训练调度：前 50% epoch 关闭因果项，后 50% epoch 线性升至 `lambda_causal=0.1`。
- mask 规模：full num mask `(10,10)` 非零 21；full cat mask `(129,129)` 非零 4534；Stage1 mask 非零 0。
- 结果：TSTR=0.7199，Best=0.7440，R2=0.6138，MSE=0.2844，MAE=0.2018，WD=0.2101，JS=0.0089。

### 4. 当前判断

- Opt-C 相比原完全体 `macro_soft_2024_v2` 有明显恢复：TSTR 从 0.7083 提升到 0.7199，WD 从 0.4158 降到 0.2101，JS 从 0.0095 降到 0.0089。
- Opt-C 仍未超过 `ablation_no_causal_v2` 的 TSTR=0.7394，差距约 0.0195。
- 论文叙事上，Opt-C 更适合作为“可解释宏观因果约束”的候选版本；如果主指标优先，当前最强域内模型仍是无因果分层消融。
- 下一步建议：保留 Opt-C 作为解释性主线，同时准备 2025 迁移测试，观察稀疏宏观因果是否在跨年泛化上缩小或反超无因果消融。
