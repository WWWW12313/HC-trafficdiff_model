# 2026-05-18 实验日志

## CausalDiffTab v2 宏观稀疏因果图 + 退火 λ

- 背景：v3 λ 灵敏度实验显示，简单降低 `lambda_causal` 未超过 `ablation_no_causal_v2`；`lambda=0.05` 改善 WD/JS，但 TSTR 仍被压制。
- 新方向：按宏观机制减少因果边，仅保留 `time -> weather`、弱 `geo -> weather`、强 `geo -> road context`、`weather/road/vehicle/time-light proxy -> crash type`、`crash type/road/vehicle/weather -> injury outcome`。
- 新增 mask：`causal_masks_macro_sparse`，由 `pipeline/make_macro_sparse_masks_v2.py` 直接生成，避免旧 domain rule expander 自动补太多边。
- 新增训练调度：`causal_start_frac=0.5`，`causal_warmup_frac=0.5`，即前半程关闭因果正则，后半程线性升至 `lambda_causal=0.1`。
- 新实验：`macro_sparse_anneal_v2`，目标是验证“少边宏观因果 + 延迟退火”能否兼顾 TSTR 与分布保真度。
